## Dependencies

The following Python packages should be installed prior to first migration:

```
Pillow
django-mptt
django-tagging
beautifulsoup4
mots-vides
django-contrib-comments
django-blog-zinnia
zinnia-wysiwyg-wymeditor
```

Each package may be installed via:

```
$ pip install [package-name]
```
